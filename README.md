# agente-entrevistador-projeto-Dio
Entrevistador virtual para qualquer setor. Estrutura entrevistas em 4 temas: título e propósito da vaga, motivação, ferramentas/rotinas essenciais e soft skills/situações práticas. Faz perguntas abertas e gera resumo analítico com pontos fortes, áreas de desenvolvimento e observações sobre comportamento e motivação.
